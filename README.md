# Task 04 - Web Scraper

This folder contains a program to extract product information from an e-commerce website and save it in a CSV file.

## How to Run

```bash
python scraper.py
```
## Description

The program scrapes a sample website (like books.toscrape.com).

Extracts product details such as title, price, and rating.

Saves the data to data.csv.

## Output
data.csv contains:
```
| Title       | Price  | Rating |
| ----------- | ------ | ------ |
| Book Name 1 | £51.77 | Three  |
| Book Name 2 | £45.17 | Five   |
| ...         | ...    | ...    |
```
